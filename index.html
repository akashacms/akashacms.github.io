AkashaCMS on GitHub

AkashaCMS is a family of Node.js packages that handles static website generation and building EPUB's.

You can learn more about AkashaCMS at - https://akashacms.com

The following are per-project websites for AkashaCMS modules hosted here on GitHub.

EPUBTools

https://akashacms.github.io/epubtools - Handles bundling EPUB eBooks
Stacked Directories

https://akashacms.github.io/stacked-directories - Handles the complex directory structures we can build with AkashaCMS, and detects changes to initiate automatic rebuild
Open Source Sample Website

https://akashacms.github.io/open-source-site - Example website that could be useful for open source software projects
